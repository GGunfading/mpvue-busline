# 公交快查

## 介绍

> 提供公交地铁路线导航、公交地铁换乘方案。

![image](./static/image/gh_c1cf07dd7cde_344-2.jpg)

## 运行

进入 clone 下来的文件目录，执行下面命令：

``` bash
# 安装依赖，启动实时编译服务
npm install && npm run dev
```

## 查看效果

打开微信开发者工具，选择克隆下来的目录即可查看。
